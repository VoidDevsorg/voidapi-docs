# Get User

<br />
## API Reference

### Base URL

```http
https://voidapi.rest/v1/apis/games/minecraft/users/:username
```

### Params

| Name | Type | Description | Values |
| :--- | :--- | :--- | :--- |
| `username` | `string` | The username of the user. |  |

### Headers
| Name | Type | Description |
| :--- | :--- | :--- |
| Authorization | `string` | The API key. |
