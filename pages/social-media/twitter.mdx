# Twitter API

<br />
## API Reference

### Base URL

```http
https://voidapi.rest/v1/apis/socials/twitter/:username
```

### Parameters

| Name | Type | Description |
| :--- | :--- | :--- |
| `username` | `string` | The username of the user. |

### Headers
| Name | Type | Description |
| :--- | :--- | :--- |
| Authorization | `string` | The API key. |
