# WWW Search

<br />
## API Reference

### Base URL

```http
https://voidapi.rest/v1/apis/web/engines/search
```

### Queries

| Name | Type | Description | Values |
| :--- | :--- | :--- | :--- |
| `q` | `string` | The query to search for. | any |
| `offset` | `number` | The offset to start at. | any |
| `safe` | `string` | The safe search level. | `off`, `moderate`, `strict` |
| `mkt` | `string` | The language to search in. | any like `en-US` |

### Headers
| Name | Type | Description |
| :--- | :--- | :--- |
| Authorization | `string` | The API key. |
