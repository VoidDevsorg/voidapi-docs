# YouTube Search

<br />
## API Reference

### Base URL

```http
https://voidapi.rest/v1/apis/web/engines/videos
```

### Queries

| Name | Type | Description |
| :--- | :--- | :--- |
| `q` | `string` | The query to search for. |

### Headers
| Name | Type | Description |
| :--- | :--- | :--- |
| Authorization | `string` | The API key. |
