# ImgBB Upload

<br />
## API Reference

### Base URL

```http
https://voidapi.rest/v1/apis/images/upload
```

### Body

| Name | Type | Description | Values |
| :--- | :--- | :--- | :--- |
| `images` | `Array` | The images to search. | `Base64[]` |

### Headers
| Name | Type | Description |
| :--- | :--- | :--- |
| Authorization | `string` | The API key. |

### Fetch
